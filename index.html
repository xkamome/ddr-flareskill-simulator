使い方

概要

DDR のフレアスキルをシミュレーションする非公式ツールです。ブックマークレットで DDR World から自分のスコアデータを取得し、新規スコアの追加・既存スコアの更新・削除によってスキル合計値がどう変わるかをシミュレーションできます。シミュレーション結果は URL で保存・共有でき、画像としてダウンロードすることもできます。

保存データは最終アクセスから 90 日で自動削除されます。「JSONとしてDL」でのバックアップを推奨します。

※ 本ツールは KONAMI および DDR World と無関係の非公式ツールです。

ブックマークレット登録

  1. 下のコードをコピーする
  2. ブラウザのブックマークを新規作成し、URL欄にそのままペーストする
  3. DDR World のフレアスキル対象楽曲ページでブックマークを実行する

インポート方法

  • ブックマークレット — KONAMI の DDR World 公式 Web サイトにログインし、フレアスキル対象楽曲ページでブックマークレットを実行。データ送信後に自動でシミュレータが開きます。
  • JSON読込 — 以前に「JSONとしてDL」で保存したファイルをインポートできます。

シミュレーション

  • 新規スコア — 新規でフレアスキルを更新した場合の、スキル表への影響をシミュレーション。
  • 既存更新 — 既存スコアのランクアップをシミュレーション。テーブルの行をクリックすると自動で選択されます。
  • 候補曲 — SOON / MID / LONG のティアで練習候補を管理。シミュレーション結果には影響しません。

SP / DP

  • ブックマークレットまたは JSON 読込時に SP / DP を判別してロックします。
  • 新規シミュレーション時はヘッダーのトグルで切り替えられます。

保存・共有

  • シミュレーションを保存 — 外部サービスのデータベースに保存し、URLが変わります。ブックマーク登録すると後からこのシミュレーションにアクセスしやすくなります。
  • フォーク — 共有されたシミュレーションは自分なりに編集することができます。元のデータは変更されず、新規のレコードとして保存されます。URLを知っている人は誰でも閲覧・フォークできます。
  • JSONとしてDL — ローカルにデータを JSON としてダウンロードします。JSON読込で復元できます。
  • 画像としてDL — スキル表・集計・コメントを含む画像を生成します。
  • 𝕏 で共有 — 現在のスキル合計とランクを X に投稿します。

目標スキル値

  • 右ペイン上部に目標値を入力すると、現在値との差分と進捗バーが表示されます。
  • プリセット（ランク境界値）から選ぶこともできます。

楽曲メモ・コメント

  • 楽曲メモ — テーブルの行をクリックすると右ペインにメモ欄が表示されます（140字以内）。
  • 全体コメント — シミュレーション全体へのコメント。画像化時に含まれます。

データの有効期限

  • 保存データは最終アクセスから 90 日で自動削除されます。
  • URL にアクセスするたびに有効期限が延長されます。
  • 期限が近づくと警告が表示されます。「JSONとしてDL」でバックアップを保存しておくことを推奨します。

データポリシー

  • ブックマークレットは KONAMI 公式の DDR World ページ上の楽曲スコアデータのみを取得します。
  • 取得する情報: 楽曲名・難易度・レベル・フレアランク・フレアスキル値・達成日
  • プレイヤー名・アカウント情報・その他の個人情報は一切取得しません。
  • 取得データは外部サービスのデータベースに保存され、有効期限経過後に自動削除されます。

📱 モバイル版の操作

  • 右下の ☰ ボタン — シミュレーターパネル（右ペイン）の開閉。スコア入力・目標値設定はここから。
  • 枠ヘッダ（CLASSIC / WHITE / GOLD）をタップ — その枠の表をトグル（折りたたみ／展開）。3枠すべて閉じるとページ先頭に戻ります。
  • CANDIDATES をタップ — 候補曲リストの折りたたみ／展開。候補曲を追加すると自動で展開されます。
  • SP / DP 切替 — シミュレーターパネル上部（右ペイン内）から変更できます。
  • UUID — シミュレーターパネル上部に短縮表示されます。
